### 作業三 => React + ChatGPT 整合
---

這是一個以 **React** 製作的前端 AI 聊天介面，  
可直接在瀏覽器中輸入 API Key 並與 AI 模型互動。  
支援嵌入至現有網站（例如個人作品集頁面），不需後端伺服器即可執行。

---

## 📘 專案簡介

此專案的目的為：
- 建立一個簡潔、可自行輸入 API Key 的聊天視窗。
- 示範如何在靜態網頁中嵌入 React 元件。
- 提供快速測試 OpenAI 或 Gemini API 回應效果的介面。
- 支援本機儲存 API Key、上下文聊天紀錄與簡易滾動機制。
- 可依網站主題（亮／暗色模式）自動調整樣式。

---

## ⚙️ 功能說明

| 功能 | 說明 |
|------|------|
| 💬 聊天功能 | 可輸入訊息與 AI 模型互動。 |
| 🔑 本機 API Key 儲存 | 使用 localStorage 暫存 API Key（僅限瀏覽器本機）。 |
| 🧠 模型切換 | 可手動輸入模型 ID，如 `gpt-5`, `gpt-4o-mini`。 |
| 🌗 主題偵測 | 自動依照網站的 `data-theme` 切換深／淺色模式。 |
| 🧩 可嵌入式 | 支援將 React 元件掛載在 `<div id="react-root"></div>`，適用任何靜態網站。 |
| 🪶 無後端依賴 | 透過前端直接呼叫 API（教學用 Demo，不建議正式部署）。 |

---

## 安裝與執行
```bash
npm install
npm start
```
